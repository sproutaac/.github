# Getting Support

## Usage questions and general help

Please don't open a GitHub issue for usage questions — issues are for bugs and feature requests.

For help using Sprout, questions about AAC, or general conversation:

- **Discord** — [join the OpenAAC community](https://discord.gg/TEH8uxh) where SLPs, families, and developers hang out
- **Email** — [hello@sproutaac.org](mailto:hello@sproutaac.org) for anything you'd rather not post publicly

## Bug reports and feature requests

[Open an issue](https://github.com/sproutaac/app/issues) on the app repo. See [CONTRIBUTING.md](CONTRIBUTING.md) for what to include.

## Security vulnerabilities

See [SECURITY.md](SECURITY.md). Please do not report vulnerabilities in public issues.
