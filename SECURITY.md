# Security Policy

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Email [hello@sproutaac.org](mailto:hello@sproutaac.org) with a description of the issue, steps to reproduce, and any relevant context. We will acknowledge your report within 72 hours and aim to release a fix within 14 days for confirmed vulnerabilities.

Given that Sprout is designed for use by children, we take security and privacy reports especially seriously. Child communication data must never be exposed, leaked, or accessible to unauthorised parties. If your finding relates to child data in any way, please flag that explicitly.

We will credit researchers who report valid vulnerabilities, unless you prefer to remain anonymous.

## Supported Versions

The app is pre-release and not yet on the App Store or Google Play. The website at sproutaac.org is live. Security reports for both are welcome.
